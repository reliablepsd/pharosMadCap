Pharos Insights Overview

Pharos Insights is a cloud-based solution that tracks print usage as well as print costs across a company's fleet of devices. It reveals the true cost of printing and provides decision support to identify usage patterns, reduce cost, and optimize the fleet.

Pharos Insights consists of two products: Fleet Analytics and Print Analytics

Fleet Analytics

Fleet Analytics also referred to as Fleet Manager provides a comprehensive, multi-vendor view of print from all devices and reveals volume, service, and operating cost information to help you build and maintain a more efficient fleet.

The Fleet Dashboard provides a quick view of the key metrics in the print environment including total operating cost, page volume, utilization, and sustainability information. The dashboard can be used to quickly identify which manufacturers, device types, and print options are generating the most cost and then drill down to the details you need to develop effective print strategies.

The Fleet > Setup Guide section defines the steps you need to follow to configure Fleet Analytics. For more information on Fleet Analytics, refer to the Setup Guide document.

Print Analytics

Print Analytics provides insights into how print is created in your environment, including information about the user, the application, the device, and comprehensive output parameters. It also reveals opportunities to reduce costs and waste, and it helps you to discover the fleet design that best supports your people

The Print Dashboard provides a summary of key metrics for the print environment over the last 30 days. It highlights areas that require further exploration and helps to identify actions that can be taken to improve your print environment.

The Print > Setup Guide section defines the steps you need to follow to configure Print Analytics. For more information on Print Analytics, refer to the Setup Guide document.

On-premises Components

Pharos Insights employs two locally-deployed Scouts that collect fleet, print, and user information. All data is transmitted to the cloud using secure, encrypted channels

  • The Device Scout is installed on a Windows server. It continually discovers and collects information from network print devices and sends that information to the Insights server (cloud). It also runs various tasks on defined schedules; these include discovering new devices on the network and collecting meter, supply, status, and alert information. After your network print devices are discovered, they are listed in the Fleet tab within Insights.

    The Device Scout is required for Fleet Analytics.

  • Print Scouts are installed on workstations typically using a software deployment tool such as SMS. Print Scouts are deployed to collect comprehensive information about how print is created within the organization. Print Scouts on print servers (PC only) collect information on network printers and multi-function devices while Print Scouts deployed to workstations (PC and Mac) collect both network printing activity and any printing sent to locally attached devices. This information is sent to the Insights server for real-time analysis and reporting capability.

    The Print Scout is required for Print Analytics.
